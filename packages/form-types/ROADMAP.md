# Roadmap

## Feature Releases

### Short-term
- [ ] **Testing**
- [ ] **Conditional field types** - `DependsOn` type for field dependencies
- [ ] **Field group types** - Support for nested field groups
- [ ] **Repeatable field types** - Types for repeatable field groups
- [ ] **Validation schema types** - Schema-based validation types

### Medium-term
- [ ] **i18n support types** - Translation and localization types
- [ ] **Theme types** - Theming and styling configuration
- [ ] **Async validation types** - Promise-based validation rules
- [ ] **Cross-field validation types** - Types for validations that depend on multiple fields

### Long-term
- [ ] **Custom field registration** - Allow users to extend FieldConfigMap
- [ ] **Plugin system types** - Types for plugin architecture
- [ ] **State management types** - Types for form state (touched, dirty, etc.)
- [ ] **Event system types** - Types for form events and hooks
